# ECG Analyzer Web Application

A web-based ECG analysis tool built with React and Vite, designed to analyze and visualize ECG data.

## 🚀 Features

- Real-time ECG waveform visualization
- Interactive data analysis
- User-friendly interface
- Comprehensive report generation
- Multi-language support

## 🛠️ Technologies Used

- React 18
- Vite
- Tailwind CSS
- Chart.js
- React Router DOM
- Axios

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- Node.js (version 16.0 or higher)
- npm or yarn
- Git

## ⚙️ Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/ecg-analyzer.git
```

2. Navigate to the project directory
```bash
cd ecg-analyzer
```

3. Install dependencies
```bash
npm install
# or
yarn install
```

4. Start the development server
```bash
npm run dev
# or
yarn dev
```

5. Build for production
```bash
npm run build
# or
yarn build
```
